# Telas implementadas
 
## Tela de Cadastro de Usuário
 
- [Tela de Cadastro](paginas/cadastro.html)
  - Tela responsável por permitir que novos usuários se registrem no sistema. Apresenta um formulário com os campos nome completo, e-mail, telefone, senha, confirmação de senha e tipo de usuário (Cliente, Funcionário ou Administrador). É uma funcionalidade pública, acessível sem autenticação, voltada principalmente ao cadastro de novos clientes da oficina.
## Tela Principal (Painel do Cliente)
 
- [Tela Principal](paginas/principal.html)
  - Tela exibida ao cliente após o login. Apresenta um resumo do seu perfil no sistema, com cards indicando a quantidade de veículos cadastrados, o total de ordens de serviço e quantas estão em andamento. Também exibe a lista de ordens de serviço com número, veículo, tipo de serviço e status atual (Em espera, Em reparação ou Concluído), além dos veículos vinculados à conta com a opção de adicionar novos.
